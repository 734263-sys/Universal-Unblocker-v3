Theme creation guide

A guide on creating themes, and what is currently supported!

Last updated: 1/17/25

Format

All themes should have a list of css variables.

Here is an example of the default theme!

A few variables have fallback values, like if there is no --sidebar-icon-color set, it will default to rgb(255,255,255)

Even if a variable has a fallback value, it is still heavily recommended to overwrite it by including it in your theme!

Customization

I recommend using the base provided above as a template / start to all of your plugins!

I heavily recommend keeping the variables in, but your not limited, you can add any css you'd want!
